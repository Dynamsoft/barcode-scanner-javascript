# WebView Sample

This sample demonstrates how to use the `BarcodeScanner` API from the Dynamsoft Barcode Reader JavaScript SDK inside native WebView-based apps for Android and iOS.

## Structure

- `android/` - Android WebView project
- `ios/` - iOS WebView project

## Quick Start

### Android

1. Open the `android` folder in Android Studio
2. Follow the platform-specific setup instructions in the android folder
3. Build and run on an Android device or emulator

### iOS

1. Open the `ios` folder in Xcode
2. Follow the platform-specific setup instructions in the ios folder
3. Build and run on an iOS device or simulator

## Notes

- Native build environments (Android Studio / Xcode) are required
- Camera permissions must be configured in the native apps
- See the Dynamsoft documentation for WebView-specific integration details
